## Naming conventions
- Dart `camelCase` for fields, `snake_case` on the wire (`@JsonKey`).
- Pure-Dart models in the engine: plain Dart classes with `fromJson` / `toJson` / `copyWith` (no freezed codegen since this is a fresh project without `dart pub run build_runner`).
- Engine version: `3.2.0`.
- Banker's rounding everywhere except rep-range math (round-half-up).
