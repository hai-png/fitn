   341Ôćĺ### 4.2 Assessment sub-engine
   342Ôćĺ
   343ÔćĺOrchestration (`assessProfile`) runs 4 sub-assessments in order, **never short-circuiting on failure**:
   344Ôćĺ
   345Ôćĺ1. `assessBodyComposition(profile)` ÔÇö root of the dependency tree.
   346Ôćĺ2. `assessHealthRisk(profile)` ÔÇö independent of body comp.
   347Ôćĺ3. `assessMuscularPotential(profile, bodyComp.bodyFatPct)` ÔÇö **skipped** if `bodyComp == null`.
   348Ôćĺ4. `decideStrategy(profile, bodyComp.bodyFatPct)` ÔÇö **skipped** if `bodyComp == null`, defaults to `maintenance`.
   349Ôćĺ
   350ÔćĺIf any sub-assessment throws, record `"sub_name: ArgumentError: <msg>"` in `errors` and set the sub-result to null. `is_partial = errors.isNotEmpty`. The summary string is built defensively (every nullable field guarded).
   351Ôćĺ
   352Ôćĺ`proposePlan()` throws `PartialAssessmentError` if called with `is_partial == true`.
   353Ôćĺ
   354Ôćĺ#### 4.2.1 Body fat % selection (priority order)
   355Ôćĺ
   356Ôćĺ1. **User-provided** (`bodyFatPct != null`) Ôćĺ use as-is.
   357Ôćĺ2. **US Navy circumference** (Hodgdon & Beckett 1984) ÔÇö if `hasCircumferenceMeasurements`:
   358Ôćĺ   - Male: `86.010 ├Ś log10(waistIn Ôłĺ neckIn) Ôłĺ 70.041 ├Ś log10(heightIn) + 36.76`
   359Ôćĺ   - Female: `163.205 ├Ś log10(waistIn + hipIn Ôłĺ neckIn) Ôłĺ 97.684 ├Ś log10(heightIn) Ôłĺ 78.387`
   360Ôćĺ   - If `(waistIn Ôłĺ neckIn) ÔëĄ 0` (male) or `(waistIn + hipIn Ôłĺ neckIn) ÔëĄ 0` (female), return null Ôćĺ fall through to CUN-BAE.
   361Ôćĺ   - Clamp to `[2.0, 60.0]`.
   362Ôćĺ3. **CUN-BAE** (always computable):
   363Ôćĺ   - `bf = -24.988 + 0.503 ├Ś age + 1.0689 ├Ś bmi + 0.462 ├Ś (sex == female ? 1.0 : 0.0)`
   364Ôćĺ   - Clamp to `[2.0, 60.0]`.

   430Ôćĺ#### 4.2.7 Strategy decision tree
   431Ôćĺ
   432Ôćĺ**Sex-specific boundaries** (single source of truth):
   433Ôćĺ
   434Ôćĺ| Boundary           | Male | Female |
   435Ôćĺ| ------------------ | ---- | ------ |
   436Ôćĺ| cut_floor          | 10   | 18     |
   437Ôćĺ| bulk_ceiling       | 20   | 28     |
   438Ôćĺ| bulk_start         | 15   | 23     |
   439Ôćĺ| operational_lo     | 10   | 18     |
   440Ôćĺ| operational_hi     | 20   | 28     |
   441Ôćĺ| obese_threshold    | 25   | 32     |
   442Ôćĺ| recomp_excellent   | 23   | 30     |
   443Ôćĺ| recomp_good_lo     | 15   | 25     |
   444Ôćĺ| recomp_limited     | 15   | 25     |
   445Ôćĺ| skinny_fat_lo      | 12   | 20     |
   446Ôćĺ| skinny_fat_hi      | 23   | 31     |
   447Ôćĺ
   448Ôćĺ**Algorithm (first match wins):**
   449Ôćĺ
   450Ôćĺ1. **Safety override (obesity):** `bfPct >= obese_threshold`:
   451Ôćĺ   - BEGINNER + goal != fat_loss Ôćĺ `habitChangeFirst` ("start with habit changes before calorie counting")
   452Ôćĺ   - Else Ôćĺ `cut` ("prioritise fat loss")
   453Ôćĺ2. **Goal = maintenance:**
   454Ôćĺ   - If `bfPct < hormonal_floor` Ôćĺ warn about bulk.
   455Ôćĺ   - If `bfPct > operational_hi` Ôćĺ warn about cut.
   456Ôćĺ   - **Reverse-diet detection:** if `intakeLogKcal.length >= 30` AND `operational_lo <= bfPct <= operational_hi`:
   457Ôćĺ     - Trailing 7-day avg intake vs estimated TDEE (compute via RMR+TDEE pipeline; fallback `weightKg ├Ś 35`).
   458Ôćĺ     - If `recentIntake < tdee ├Ś 0.90` Ôćĺ `reverseDiet`.
   459Ôćĺ   - Else Ôćĺ `maintenance`.
   460Ôćĺ3. **Goal = strength:** if `bfPct > bulk_ceiling` Ôćĺ `cut`; else `maintenance`.
   461Ôćĺ4. **Goal = fat_loss:** if `bfPct < cut_floor` Ôćĺ `maintenance` (protect hormones); else `cut`.
   462Ôćĺ5. **Goal = muscle_gain:** if `bfPct > bulk_ceiling` Ôćĺ `cut`; else `bulk`.
   463Ôćĺ6. **Goal = recomp:** `bfPct >= recomp_excellent` Ôćĺ `recomp` (excellent); `>= recomp_good_lo` Ôćĺ `recomp` (good); else `bulk` (limited).
   464Ôćĺ7. **Auto-decide (no explicit goal override ÔÇö fallback path):**
   465Ôćĺ   - `bfPct > operational_hi` Ôćĺ `cut`
   466Ôćĺ   - `bfPct < cut_floor` Ôćĺ `bulk`
   467Ôćĺ   - Novice + skinny-fat range Ôćĺ `recomp`
   468Ôćĺ   - Novice + `bfPct < bulk_start` Ôćĺ `recomp` (milk newbie gains)
   469Ôćĺ   - `bfPct < bulk_start` Ôćĺ `bulk`
   470Ôćĺ   - Else Ôćĺ `cut`

   472Ôćĺ### 4.3 Nutrition sub-engine
   473Ôćĺ
   474Ôćĺ`buildNutritionPlan(profile, assessment, exerciseHours, exerciseIntensity, climate, weightReducedPct)`:
   475Ôćĺ
   476Ôćĺ1. **RMR** ÔÇö `KATCH_MCARDLE` if `bodyFatPct != null`, else `MIFFLIN_ST_JEOR`.
   477Ôćĺ   - Mifflin: `10 ├Ś weight + 6.25 ├Ś height Ôłĺ 5 ├Ś age + (male ? 5 : -161)`
   478Ôćĺ   - Katch-McArdle: `370 + 21.6 ├Ś lbm` (throws if `bfPct Ôłë [2, 60]`)
   479Ôćĺ   - Adjustments: `├Ś 0.95` if active deficit (cut/recomp), `├Ś 0.97` if `weightReducedPct > 0.10`.
   480Ôćĺ2. **TDEE** ÔÇö RippedBody activity factors (1.25 / 1.45 / 1.65 / 1.85 / 2.05 for sedentary / mostly_sedentary / lightly_active / active / highly_active).
   481Ôćĺ3. **Adaptive TDEE** ÔÇö if `weightLogKg` and `intakeLogKcal` are both non-empty, equal-length, and Ôëą8 entries:
   482Ôćĺ   - `observed = avgIntake Ôłĺ (╬öweight ├Ś 7700) / nDays`
   483Ôćĺ   - Sanity: `observed Ôłł [800, 7000]`, else skip.
   484Ôćĺ   - Ramp weight: `w = clamp((nDays Ôłĺ 7) / 53, 0, 1)` ÔÇö no adaptation below 7 days, full at 60+.
   485Ôćĺ   - `adaptive = w ├Ś observed + (1 Ôłĺ w) ├Ś tdee`.
   486Ôćĺ4. **Calorie targets** ÔÇö by strategy, with floor 1200 (F) / 1500 (M):
   487Ôćĺ
   488Ôćĺ| Strategy           | Computation                                                                                       |
   489Ôćĺ| ------------------ | ------------------------------------------------------------------------------------------------- |
   490Ôćĺ| cut                | `weeklyLoss = weight ├Ś rate`; `dailyDeficit = weeklyLoss ├Ś 1100`; `target = tdee Ôłĺ dailyDeficit` |
   491Ôćĺ| bulk               | `monthlyGain = weight ├Ś rate`; `dailySurplus = monthlyGain ├Ś 330`; `target = tdee + dailySurplus` |
   492Ôćĺ| maintenance        | `target = tdee`                                                                                   |
   493Ôćĺ| recomp             | `target = tdee ├Ś (1 Ôłĺ deficitPct)` where deficitPct = 0.15/0.05/0.0 by recomp potential           |
   494Ôćĺ| reverse_diet       | Weekly ladder: `+50/+100/+150 kcal/wk` by aggressiveness, until target reached                    |
   495Ôćĺ| habit_change_first | Same as maintenance                                                                               |
   496Ôćĺ
   497Ôćĺ**Cut rate** (priority): explicit `cutRateTier` Ôćĺ BF%-threshold table Ôćĺ default 0.75% BW/week. Hard cap 1.0% BW/week.
   498Ôćĺ
   499Ôćĺ**Cut rate by BF% threshold** (first match):
   500Ôćĺ
   501Ôćĺ| Sex    | Thresholds (BF%) | Rate (% BW/week)   |
   502Ôćĺ| ------ | ---------------- | ------------------ |
   503Ôćĺ| Male   | [25, 20, 15]     | [1.00, 0.75, 0.50, 0.50] |
   504Ôćĺ| Female | [32, 28, 22]     | [1.00, 0.75, 0.50, 0.50] |
   505Ôćĺ
   506Ôćĺ**Bulk rate** (priority): explicit `bulkAggressiveness ├Ś trainingStatus` table Ôćĺ default by status (beginner 2%/mo, novice 1.5%/mo, intermediate 1%/mo, advanced 0.5%/mo).
   507Ôćĺ
   508Ôćĺ**Bulk weekly rate tiers** (% BW/week):
   509Ôćĺ
   510Ôćĺ| Aggressiveness     | Beginner/Novice | Intermediate | Advanced |
   511Ôćĺ| ------------------ | --------------- | ------------ | -------- |
   512Ôćĺ| conservative       | 0.0020          | 0.0015       | 0.0010   |
   513Ôćĺ| happyMedium        | 0.0050          | 0.00325      | 0.0015   |
   514Ôćĺ| aggressive         | 0.0080          | 0.00575      | 0.0035   |
   515Ôćĺ| veryAggressive     | 0.0100          | 0.00800      | 0.0060   |
   516Ôćĺ
   517Ôćĺ5. **Macros:**
   518Ôćĺ   - **Protein:**
   519Ôćĺ     - Obese override (`bfPct >= obese_threshold`): `1 g ├Ś heightCm`.
   520Ôćĺ     - BF% known: `lbm_lb ├Ś 1.14` (cut) or `├Ś 1.0` (else).
   521Ôćĺ     - BF% unknown: `target_weight_lb ├Ś 1.0` (cut, target = 90% current) or `weight_lb ├Ś 0.73` (else).
   522Ôćĺ     - Diet boost: vegan ├Ś1.20 (ceil), vegetarian ├Ś1.10 (ceil).
   523Ôćĺ   - **Fat:** midpoint of strategy range (deficit/recomp: 15-25%; surplus/maintenance/reverse: 20-30%). Floor: `max(40, weight_lb ├Ś 0.25)`. Saturated ceiling: 10% of calories.
   524Ôćĺ   - **Carbs:** remainder = `targetKcal Ôłĺ proteinKcal Ôłĺ fatKcal`, clamped to Ôëą0.
   525Ôćĺ6. **Hydration** (multi-step FatCalc):
   526Ôćĺ   - Base: `weightKg ├Ś 0.030 L`
   527Ôćĺ   - Sex: +0.3 L (male)
   528Ôćĺ   - Exercise: `hours ├Ś sweatRate / 1000` (light 300, moderate 500, intense 800 mL/hr)
   529Ôćĺ   - Climate multiplier (cold 0.95, temperate 1.0, hot 1.3, hot_humid 1.4) applied to exercise component only
   530Ôćĺ   - Pregnancy +0.3 L, breastfeeding +0.7 L (throws if male)
   531Ôćĺ   - Soft ceiling 5.0 L (hyponatremia protection)
   532Ôćĺ7. **Micronutrients:** fiber = `14 g ├Ś targetKcal / 1000`. Fruit/veg cups: 2/2 (ÔëĄ2000 kcal), 3/3 (ÔëĄ3000), 4/4 (ÔëĄ4000+).
   533Ôćĺ8. **Timeline:**
   534Ôćĺ   - Cut: `max(4, floor(kgToLose / weeklyRateKg) + 4)` where target BF = `operational_lo + 2`.
   535Ôćĺ   - Bulk: `max(12, floor((targetGainKg / monthlyRateKg) ├Ś 4.348) + 4)` where targetGain = 5% BW.
   536Ôćĺ   - Recomp: 12 weeks. Maintenance: 12. Habit-change: 8. Reverse-diet: parse from notes, default 8.

   652Ôćĺ
   653Ôćĺ`buildMealPlan(profile, assessment, nutrition, mealFrequency, cuisinePreference, allergensToAvoid, excludedIngredients, includePrePostWorkout)`:
   654Ôćĺ
   655Ôćĺ1. **Compute slot targets** from daily macros ├Ś meal allocation:
   656Ôćĺ
   657Ôćĺ| Frequency | Allocation                                              |
   658Ôćĺ| --------- | ------------------------------------------------------- |
   659Ôćĺ| 2 meals   | LUNCH 0.50, DINNER 0.50 (no breakfast ÔÇö intentional)   |
   660Ôćĺ| 3 meals   | BREAKFAST 0.30, LUNCH 0.35, DINNER 0.35                 |
   661Ôćĺ| 4 meals   | BREAKFAST 0.25, LUNCH 0.30, DINNER 0.30, SNACK 0.15    |
   662Ôćĺ| 5 meals   | BREAKFAST 0.20, LUNCH 0.25, DINNER 0.30, SNACK 0.15, SNACK 0.10 |
   663Ôćĺ
   664Ôćĺ2. **Determine training days** for the 7-day week:
   665Ôćĺ   - 2d Ôćĺ [1, 4], 3d Ôćĺ [1, 3, 5], 4d Ôćĺ [1, 2, 4, 5], 5d Ôćĺ [1, 2, 3, 4, 5], 6d Ôćĺ [1, 2, 3, 4, 5, 6].
   666Ôćĺ3. **For each of 7 days ├Ś N slots:** allocate meal via `scoreRecipeForSlot` Ôćĺ pick top Ôćĺ scale Ôćĺ fill gap.
   667Ôćĺ4. **Track recipe usage** (rolling 3-day and 7-day windows) for variety scoring.
   668Ôćĺ
   669Ôćĺ**Diet tag derivation:**
   670Ôćĺ- omnivore Ôćĺ `"OMNI"`
   671Ôćĺ- vegan Ôćĺ `"VEGAN"`
   672Ôćĺ- vegetarian Ôćĺ `"VEGAN"` (recipe DB has no separate vegetarian tag ÔÇö vegetarians get vegan recipes)
   673Ôćĺ
   674Ôćĺ**Recipe scoring** (9 weighted components, each 0-100):
   675Ôćĺ
   676Ôćĺ| Component        | Weight | Tight band | Loose band |
   677Ôćĺ| ---------------- | ------ | ---------- | ---------- |
   678Ôćĺ| kcal_match       | 26     | ┬▒20%       | ┬▒40%       |
   679Ôćĺ| protein_match    | 22     | ┬▒15%       | ┬▒30%       |
   680Ôćĺ| carb_match       | 13     | ┬▒20%       | ┬▒40%       |
   681Ôćĺ| fat_match        | 9      | ┬▒25%       | ┬▒50%       |
   682Ôćĺ| diet_match       | 13     | ÔÇö          | ÔÇö          |
   683Ôćĺ| goal_fit         | 4      | ÔÇö          | ÔÇö          |
   684Ôćĺ| fiber_match      | 4      | ┬▒50%       | ┬▒100%      |
   685Ôćĺ| variety_bonus    | 4      | ÔÇö          | ÔÇö          |
   686Ôćĺ| cuisine_match    | 5      | ÔÇö          | ÔÇö          |
   687Ôćĺ
   688Ôćĺ**Hard exclusions** (score = 0):
   689Ôćĺ- Allergen violation
   690Ôćĺ- Excluded ingredient present
   691Ôćĺ- Diet mismatch
   692Ôćĺ
   693Ôćĺ**Allergen scanning** (best-in-class ÔÇö plant-qualifier suppression):
   694Ôćĺ- Normalize allergen aliases (`tree_nuts` Ôćĺ `nuts`, `crustacean(s)` Ôćĺ `shellfish`).
   695Ôćĺ- Build word-boundary regexes for each allergen's keywords.
   696Ôćĺ- Sanitize ingredients: blank out plant-named phrases (eggplant, butter lettuce, coconut milk, etc. ÔÇö full list in ┬ž9.2) before scanning for `dairy` and `eggs`.
   697Ôćĺ- For each `dairy`/`eggs` match, check 25-char context before for plant qualifiers (almond, soy, oat, coconut, etc. ÔÇö full list in ┬ž9.2). If found, skip the match.
   698Ôćĺ- Other allergens scan the unsanitized string.
   699Ôćĺ
   700Ôćĺ**Recipe scaling:**
   701Ôćĺ- `scaleFactor = clamp(targetKcal / recipeKcal, 0.7, 1.5)` (1.0 if within ┬▒10%).
   702Ôćĺ- Scaled nutrition = per-serving ├Ś scaleFactor.
   703Ôćĺ
   704Ôćĺ**Filler system** (closes macro gap after scaling):
   705Ôćĺ
   706Ôćĺ| Priority | Type     | Trigger (`gap >`) | OMNI options                                           | VEGAN* options                                |
   707Ôćĺ| -------- | -------- | ----------------- | ------------------------------------------------------ | --------------------------------------------- |
   708Ôćĺ| 1        | Protein  | 5 g               | Whey, Greek yogurt, egg white, cottage cheese, chicken | Tofu, tempeh, pea/soy protein, lentils        |
   709Ôćĺ| 2        | Carb     | 5 g               | White/brown rice, oats, banana, whole wheat bread, quinoa, sweet potato | (same)                                        |
   710Ôćĺ| 3        | Fat      | 3 g               | Olive oil, almonds, peanut butter, walnuts, avocado    | (same)                                        |
   711Ôćĺ| 4        | Veg      | 3 g (main meals only) | Broccoli, spinach, mixed greens, bell pepper, asparagus, green beans, carrots | (same)                                        |
   712Ôćĺ
   713Ôćĺ\* VEGAN prefix match ÔÇö `VEGAN_ETHIOPIAN` resolves to VEGAN list.
   714Ôćĺ
   715ÔćĺFiller grams: `(targetG / food.per100g) ├Ś 100`, capped at `serving ├Ś {protein:4, carb:3, fat:3}`, min serving = `serving ├Ś 0.5`. Veg: fixed `[80g, 200g]`.
   716Ôćĺ
   717Ôćĺ**Allergen Ôćĺ filler exclusion map:**
   718Ôćĺ- dairy Ôćĺ Whey, Greek Yogurt, Cottage Cheese, Milk, Cheddar
   719Ôćĺ- eggs Ôćĺ Egg White, Eggs
   720Ôćĺ- gluten Ôćĺ Whole Wheat Bread, Oats
   721Ôćĺ- nuts Ôćĺ Almonds, Walnuts
   722Ôćĺ- peanuts Ôćĺ Peanut Butter
   723Ôćĺ- soy Ôćĺ Tofu, Tempeh, Soy Protein, Edamame
   724Ôćĺ
   725Ôćĺ**Swap system:**
   726Ôćĺ- **Recipe swaps:** alternatives sharing Ôëą1 meal_type, diet-compatible, kcal within ┬▒20%, no allergens, cuisine-matching if specified. Sorted by kcal closeness, limited to 5.
   727Ôćĺ- **Ingredient swaps:** static substitution database (proteins, carbs, Ethiopian ingredients, fats, dairy alternatives, vegetables). Longest-key-first matching, plant-named-phrase protection.
   728Ôćĺ
   729Ôćĺ**Recipe database** (~400 recipes from 3 JSON files merged at load):
   730Ôćĺ- `recipe_database.json` (79 curated) ÔÇö `is_curated=true`, notes get `[curated]` tag.
   731Ôćĺ- `recipe_database_uncurated.json` (~305) ÔÇö `is_curated=false`. ID collisions with curated Ôćĺ rename to `U<id>`.
   732Ôćĺ- `pre_post_workout_recipes.json` (16) ÔÇö 4 diets ├Ś 2 meal types ├Ś 2 kcal bins. `nutrition_source="calculated"`, `source="engine-generated"`.
   733Ôćĺ
   734Ôćĺ**Load-time sanitization:**
   735Ôćĺ- VEGAN-tagged recipe with meat/dairy/egg ingredients Ôćĺ append `[diet-warning: tagged VEGAN but ingredients contain meat/dairy/egg ÔÇö likely a curation error]`.
   736Ôćĺ- Stated kcal vs macro-derived kcal (P├Ś4 + C├Ś4 + F├Ś9) > 10% off Ôćĺ append `[kcal-warning: ...]`.
   737Ôćĺ
   738Ôćĺ**Meal name templates** (cycled by day for variety):
   739Ôćĺ- 5-meal canonical order: BREAKFAST, SNACK, LUNCH, SNACK, DINNER (interleaved ÔÇö NOT snacks at end).
